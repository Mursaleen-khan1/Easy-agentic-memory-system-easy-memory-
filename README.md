## 🛠 About the Application  

The Standalone Agentic Memory Tool is a simple program that works alongside language models (LLMs) you might use for writing or research. It helps these models remember previous conversations and search through a large amount of information quickly.  

This tool has two main functions:  
- **On-Demand RAG**: This lets you search a big database of stored knowledge instantly when you need it.  
- **Infinite Auto-Memory**: By using the `/save` command, it remembers your conversations forever.  

Together, these features help turn any text generator into a research assistant that learns as you go.

## 📂 Managing Saved Data  

The tool saves your information in files on your computer. To manage these files:  

- Look for a folder named `EasyMemoryData` in your Documents or Program Files.  
- You can back up this folder by copying it to another drive or cloud service.  
- To clear old data, you can delete files inside this folder. Be careful, as this will remove saved conversations permanently.
